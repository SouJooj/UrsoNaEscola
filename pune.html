PUNE – Projeto Urso Na Escola

Ciência, educação e tecnologia conectadas através dos tardígrados

O que é o PUNE?

O Projeto Urso Na Escola (PUNE) é uma iniciativa educacional que utiliza os tardígrados — conhecidos como “ursos d’água” — para integrar o ensino de biologia com tecnologia.

A proposta incentiva a participação ativa de estudantes e professores, promovendo a ciência cidadã e aproximando o conhecimento científico da realidade escolar.

Objetivos do Projeto

  • ✔ Democratizar o acesso ao conhecimento científico
  • ✔ Criar uma plataforma interativa de estudo
  • ✔ Incentivar a colaboração entre alunos e professores
  • ✔ Desenvolver um banco de dados educacional
  • ✔ Promover a ciência dentro das escolas
  • ✔ Expandir o conhecimento sobre tardígrados
